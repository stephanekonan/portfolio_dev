## 🚀 Pour bien démarrer

Clonez ce dépôt sur votre machine locale avec Git.

```scheme
git clone https://github.com/stephanekonan/portfolio_dev
cd portfolio_dev
```

| Commande         | Action                                         |
| :--------------- | :--------------------------------------------- |
| `pnpm install`   | Installe les dépendances                       |
| `pnpm dev`       | Démarre le serveur local sur `localhost:4321`  |
| `pnpm build`     | Génère le site de production dans `./dist/`    |
| `pnpm preview`   | Prévisualise le build localement avant déploiement |
